Finance Tracker

A web & mobile app to track income, expenses, and savings efficiently.

Project Overview

Finance Tracker helps users monitor their financial activity in real-time. Users can add income and expense transactions, categorize them, and visualize trends using dynamic charts.

Features

  • User authentication and registration.
  • Category management for income & expenses.
  • Dynamic charts showing monthly trends.
  • Recent transaction tracking with proper history logs.
  • Responsive design for desktop and mobile.

Technologies Used

HTML, CSS, JavaScript, Firebase, MySQL, Chart.js

Challenges & Learnings

During development, integrating dynamic charts and ensuring data persistence across sessions were the most challenging tasks. I learned to effectively manage state and optimize database queries.
